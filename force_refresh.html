🔄 强制刷新 - 清除浏览器缓存

问题说明： 植物图片无法正常显示，可能是因为浏览器缓存了旧的图片映射文件。

📋 步骤 1: 强制刷新当前页面

使用以下快捷键强制刷新页面，忽略缓存：

Windows: Ctrl + F5 或 Ctrl + Shift + R
Mac: Cmd + Shift + R
或者点击浏览器的刷新按钮，按住 Shift 键

🗑️ 步骤 2: 清除浏览器缓存

如果强制刷新不起作用，请手动清除浏览器缓存：

Chrome/Edge: F12 → Application → Storage → Clear storage → Clear site data
Firefox: F12 → Storage → Clear All
Safari: Develop → Empty Caches

⚠️ 步骤 3: 重新访问主页面

清除缓存后，重新访问语言页面：

英文页面 | 中文页面 | 日文页面

✅ 验证修复

修复成功后，你应该能看到：

  • 背景图片完全覆盖页面
  • 植物卡片显示正确的详细图片（不是通用图标）
  • 浏览器控制台没有404错误
